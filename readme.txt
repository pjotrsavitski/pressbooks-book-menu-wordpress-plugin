Makes sure that Pressbooks book side menu is open by default.

== Description ==

Adds a JavaScript solution that will open the book side menu on page load.

== Installation ==

1. Upload `pressbooks-book-menu` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
